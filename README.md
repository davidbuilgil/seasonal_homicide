# seasonal_homicide
Data and analytic code used in a study to examine seasonal trends in homicide in Colombia.

Source homicide data: https://www.datos.gov.co/Seguridad-y-Defensa/HOMICIDIO/m8fd-ahd9/about_data

Source precipitation data: https://data.humdata.org/dataset/col-rainfall-subnational

Source shp file: https://www.arcgis.com/home/item.html?id=77efd7b665f54d648e60c1872fb40afa
